> For judges: this document clarifies the language policy compliance of the LabOps Agent demo video and submission materials.

## Official Statement

The Slack workspace UI text (button labels, alerts, status messages) is in Spanish, as LabOps Agent targets the Argentine clinical laboratory market.

**Key UI translations:**
- "Ver proyección" → "View forecast"
- "Ordenar reactivo" → "Order reagent"
- "Asignar al equipo" → "Assign to team"
- "CRÍTICO" → "CRITICAL"

All functionality, features, and architecture are fully narrated and explained in English throughout the demo video, in compliance with the hackathon's English-language submission requirement.
